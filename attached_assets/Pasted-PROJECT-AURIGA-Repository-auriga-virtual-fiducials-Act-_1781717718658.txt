PROJECT AURIGA

Repository: auriga-virtual-fiducials

Act as a Principal Computer Vision Engineer, Robotics Researcher, Monocular Geometry Specialist, and Accessibility Systems Architect.

Your task is to build the production-grade implementation of Auriga's Virtual Fiducial Engine.

CONTEXT

Auriga is an offline-first assistive navigation ecosystem.

The Virtual Fiducial system is a core research component.

The goal is to estimate distance from a single RGB camera using explainable geometric methods.

This repository must integrate with:

* auriga-constitution
* auriga-observatory
* auriga-data-factory

The engine must consume approved datasets generated by the Data Factory.

MISSION

Build a robust Virtual Fiducial framework capable of:

1. Estimating distance from monocular imagery.
2. Producing confidence estimates.
3. Supporting calibration workflows.
4. Supporting synthetic validation.
5. Operating entirely offline.

IMPLEMENTATION REQUIREMENTS

Create modules for:

* dataset ingestion
* calibration
* distance estimation
* confidence scoring
* evaluation
* synthetic validation
* observability

Support:

* ArUco-based calibration
* future non-marker virtual fiducials
* camera-specific calibration profiles

The system must distinguish:

* measured distance
* estimated distance
* confidence interval
* uncertainty estimate

VALIDATION

Implement:

* MAE
* RMSE
* MAPE
* calibration drift detection

Generate reports automatically.

SELF-VALIDATION

The agent must:

* test against approved datasets
* generate adversarial cases
* test lighting variation
* test reflective environments
* test partial marker visibility

Known unresolved issues must become Research Debt entries.

OUTPUTS

Generate:

* production code
* tests
* calibration pipeline
* evaluation pipeline
* CLI tools
* documentation
* validation reports

Do not proceed beyond distance estimation.

Do not implement navigation.

Do not implement world modelling.

This repository's sole purpose is Virtual Fiducial estimation.
